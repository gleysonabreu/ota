### Sobre o projeto

O TypeWeather é uma aplicação web para consultar dados meteorológicos e previsões do tempo. Com ele, é possível acessar informações de todas as cidades do mundo.

### Tecnologias

- Typescript
- Next.js
- Tailwindcss
- SWR
- Headlessui
